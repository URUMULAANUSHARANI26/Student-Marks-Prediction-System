# 🎓 Student Marks Prediction System

## 📌 Overview

The **Student Marks Prediction System** is a Machine Learning project that predicts a student's performance based on various input factors. This project helps in understanding how different features influence student marks and provides insights for improvement.

---

## 🚀 Live Demo

👉 https://student-marks-prediction-system-fw9u2g2fqxsctenlvpccsz.streamlit.app/

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/URUMULAANUSHARANI26/Student-Marks-Prediction-System.git
cd Student-Marks-Prediction-System
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run app.py
```

---

## 📈 How It Works

1. User inputs student details
2. Data is processed
3. Trained ML model predicts marks
4. Result is displayed on the web app
